<h2>📌 Overview</h2>
<p>
SplitShare is a modern web application designed to simplify expense management among groups.
Track, split, and manage shared expenses with ease in a clean and user-friendly interface.
</p>

<h2>🚀 Features</h2>

<ul>
  <li>🔐 User Authentication (Signup & Login)</li>
  <li>☑️ Terms & Conditions validation</li>
  <li>👥 Create & manage groups</li>
  <li>🧑‍🤝‍🧑 Member avatars</li>
  <li>📊 Expense tracking & splitting</li>
  <li>🔔 Real-time updates (Firebase)</li>
  <li>⚡ Fast & responsive UI</li>
</ul>

<hr/>

<h2>🧑‍💻 Tech Stack</h2>

<ul>
  <li><b>Frontend:</b> React + Vite</li>
  <li><b>Styling:</b> Tailwind CSS</li>
  <li><b>Backend/DB:</b> Firebase</li>
  <li><b>Authentication:</b> Firebase Auth</li>
  <li><b>AI Integration:</b> Gemini API</li>
  <li><b>Deployment:</b> Render</li>
</ul>

<h2>⚙️ Installation</h2>

<pre><code>
git clone https://github.com/your-username/splitshare.git
cd splitshare
npm install
</code></pre>

<hr/>

<h2>▶️ Run Locally</h2>

<pre><code>
npm run dev
</code></pre>

<hr/>

<h2>🌐 Deployment (Render)</h2>

<ol>
  <li>Push your project to GitHub</li>
  <li>Go to <b>https://render.com</b></li>
  <li>Create a <b>New Web Service</b></li>
  <li>Connect your GitHub repo</li>
  <li>Set build command:
    <pre><code>npm install && npm run build</code></pre>
  </li>
  <li>Set start command:
    <pre><code>npm run preview</code></pre>
  </li>
  <li>Deploy 🚀</li>
</ol>
